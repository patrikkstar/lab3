# Задание 1. Реализация методов фильтрации



В лабораторной работе были реализованы методы обработки временных рядов:



1. Экспоненциальное сглаживание

2. Фильтр Калмана

3. Фильтр Савицкого–Голея

4. Вейвлет-преобразование



Экспоненциальное сглаживание помогает уменьшить влияние случайных колебаний за счет взвешенного усреднения предыдущих наблюдений.



Фильтр Калмана это рекурсивный алгоритм оценки состояния системы при наличии шумов измерения.



Фильтр Савицкого–Голея выполняет локальную аппроксимацию сигнала полиномами и помогает сохранять форму сигнала при сглаживании.



Вейвлет-преобразование основано на разложении сигнала на различные частотные компоненты и удалении шумовой составляющей.
